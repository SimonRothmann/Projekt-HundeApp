# Design Vision

CanisTrack soll sich an modernen Premium-SaaS-Produkten orientieren.

Referenzen:

- Apple Fitness
- Garmin Connect
- Linear
- Notion
- Vercel
- Strava

Das Design muss vermitteln:

- Vertrauen
- Präzision
- Fortschritt
- Gemeinschaft
- Naturverbundenheit

---

# Grundprinzipien

## Mobile First

Primäre Nutzung:

Smartphone

Situation:

- Hundeplatz
- draußen
- eine Hand frei
- Handschuhe möglich

Daher:

- große Bedienelemente
- wenige Klicks
- klare Navigation
